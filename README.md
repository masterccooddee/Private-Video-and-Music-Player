# Private-Video-and-Music-Player


# 影音資料庫前端（React + Vite）

本專案為一個影音撥放器，並使用 React 搭配 Vite 建立的影音資料庫前端介面

---

## 🚀 快速啟動

### 1️⃣ 安裝相關套件
```bash
npm install
```

### 2️⃣ 啟動開發伺服器
```bash
npm run dev
```

### 3️⃣ 打開瀏覽器
開啟瀏覽器並前往：
```
http://localhost:3000
```
